# CIGALE

Classification Intelligente et Gestion des Arthropodes et de L'Entomofaune

## Développement

### MacOS / Linux / Windows (WSL)

1. Installer Volta
   ```
   curl https://get.volta.sh | bash
   ```

2. Installer les dépendances
    ```
    npm i
    ```

3. Lancer le serveur de développement
    ```
    npm dev
    ```
